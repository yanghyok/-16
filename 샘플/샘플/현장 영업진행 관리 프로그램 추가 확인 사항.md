# 현장 영업진행 관리 프로그램 — 추가 확인 사항

> 요청자 답변 후에도 불명확하게 남은 항목입니다. 개발 착수 전 재확인 필요.

---

**1. 현장코드 지역 코드표 (Q5)**
- 현장코드 형식이 "지역 + 일련번호 3자리"로 확정되었으나, 지역 코드의 구체적인 체계가 미정입니다.
- 확인 필요: 지역별 코드값 목록 (예: 서울=SE, 부산=BS, 경기=GG 등)

---

**2. SPEC OUT 확정 후 수정 가능 여부 (Q12)**
- 담당자가 확정 처리한다는 것은 확인되었으나, 확정 이후 내용 수정 허용 여부가 미정입니다.
- 확인 필요: 확정 후 수정 가능 여부, 가능하다면 수정 권한자(담당자 본인 / 관리자)

---

**3. 영업지원 권한 범위 (Q19)**
- 영업담당·관리자는 전체 수정 가능으로 확정되었으나, 영업지원의 권한 범위가 미정입니다.
- 확인 필요: 영업지원이 조회만 가능한지, 일부 수정도 가능한지
