Section 02

개발 배경 및 목적

현장 영업 과정에서 견적·특가·출고 정보가 분산 관리되어 현황 파악이 어려운 상황입니다. 이를 해소하기 위해 자동 견적 → 현장 등록 → 승인특가 흐름을 단일 시스템으로 통합하고, 현장별 영업 진행 현황을 실시간으로 관리할 수 있는 프로그램을 개발합니다.

Section 04

주요 개발 항목

4-1. 현장코드 자동 부여

  • 형식: 지역 코드 + 일련번호 3자리
  • 담당자 변경(인수인계) 시 신규 코드 발급
  • 등록 권한: 영업담당, 영업지원

4-2. 승인특가 연동

  • 승인특가 작성 시 현장코드 입력 → 현장관리 프로그램 자동 연동
  • 현장관리 프로그램에서 승인특가 작성 여부, 특가금액, 출고금액 확인 가능
  • 금액 기준 시점: 승인 시점

4-3. SPEC OUT 처리

  • SPEC OUT 정의: SPEC 수주 실패 또는 취소된 현장
  • SPEC OUT 확정 시 대체 메이커 및 대체사유 직접 입력 필수
  • 확정 처리 권한: 담당자

4-4. 현장관리 써칭 기능

번호기능
①기간별 현장 집계내역
②기간별 현장 출고내역 및 금액
③기간별 SPEC OUT 현장 내역
  • 별도 엑셀 다운로드·인쇄 기능 미포함

4-5. 관공서 영업PJ 보고양식

  • 조회 항목: 현장명, 진행단계, 담당자, 특가 여부, 출고금액
  • 출력 형태: 화면 조회
