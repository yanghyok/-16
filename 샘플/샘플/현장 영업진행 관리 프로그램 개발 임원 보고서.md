# 현장 영업진행 관리 프로그램 개발 보고

---

## 1. 보고 개요

| 항목 | 내용 |
|------|------|
| 과제명 | 현장 영업진행 관리 프로그램 개발 |

---

## 2. 개발 배경 및 목적

현장 영업 과정에서 견적·특가·출고 정보가 분산 관리되어 현황 파악이 어려운 상황입니다.
이를 해소하기 위해 **자동 견적 → 현장 등록 → 승인특가** 흐름을 단일 시스템으로 통합하고,
현장별 영업 진행 현황을 실시간으로 관리할 수 있는 프로그램을 개발합니다.

---

## 3. 확정된 업무 흐름

```
① 자동 견적 생성        공장도가 기준, 할인율 적용하여 자동 산출
        ↓
② 현장 등록             현장코드 자동 부여 (지역 코드 + 일련번호 3자리)
        ↓
③ 승인특가 작성         현장코드 연동, 특가금액 승인 시점 기준 관리
        ↓
④ 출고 및 현황 관리     현장별 출고내역·금액 조회
```

> 자동 견적을 거치지 않을 경우 현장 등록은 가능하나 승인특가 작성 불가

---

## 4. 주요 개발 항목

### 4-1. 현장코드 자동 부여
- 형식: 지역 코드 + 일련번호 3자리
- 담당자 변경(인수인계) 시 신규 코드 발급
- 등록 권한: 영업담당, 영업지원

### 4-2. 승인특가 연동
- 승인특가 작성 시 현장코드 입력 → 현장관리 프로그램 자동 연동
- 현장관리 프로그램에서 승인특가 작성 여부, 특가금액, 출고금액 확인 가능
- 금액 기준 시점: 승인 시점

### 4-3. SPEC OUT 처리
- SPEC OUT 정의: SPEC 수주 실패 또는 취소된 현장
- SPEC OUT 확정 시 대체 메이커 및 대체사유 직접 입력 필수
- 확정 처리 권한: 담당자

### 4-4. 현장관리 써칭 기능

| 번호 | 기능 |
|------|------|
| ① | 기간별 현장 집계내역 |
| ② | 기간별 현장 출고내역 및 금액 |
| ③ | 기간별 SPEC OUT 현장 내역 |

- 별도 엑셀 다운로드·인쇄 기능 미포함

### 4-5. 관공서 영업PJ 보고양식
- 조회 항목: 현장명, 진행단계, 담당자, 특가 여부, 출고금액
- 출력 형태: 화면 조회

### 4-6. 사용자 권한
| 구분 | 권한 |
|------|------|
| 영업담당 | 전체 조회·수정 |
| 관리자 | 전체 조회·수정 |
| 영업지원 | 추가 확인 중 |

| # | 항목 | 확인 필요 내용 |
