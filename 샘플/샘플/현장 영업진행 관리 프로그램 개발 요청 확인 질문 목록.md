# 현장 영업진행 관리 프로그램 — 요청자 확인 질문 목록

---

### 1단계. 전체 업무 흐름 확인

**Q1.** 업무 순서를 아래 중 어느 것으로 확정하시겠습니까?
- (A) 자동 견적 → 현장 등록 → 승인특가 작성
- (B) 승인특가 작성 → 현장 자동 등록
- (C) 두 경로 모두 허용 (자동 견적 경로 / 승인특가 선작성 경로)

> **답변: (A) 자동 견적 → 현장 등록 → 승인특가 작성**

**Q2.** "자동 견적"은 어떤 데이터를 기반으로 생성됩니까? 기존 시스템에서 가져오는 데이터가 있습니까?

> **답변: 공장도가 기준, 할인율 적용**

---

### 2단계. 현장 등록

**Q3.** 현장 등록 시 입력해야 할 필수 항목 목록을 알 수 있습니까?
(예: 현장명, 주소, 담당자, 발주처 등)

> **답변: 현장코드**

**Q4.** 현장 등록을 할 수 있는 사용자 범위는 어떻게 됩니까?

> **답변: 영업담당 및 영업지원**

---

### 3단계. 현장코드 자동 부과

**Q5.** 현장코드의 형식과 자릿수를 정해 주실 수 있습니까?

> **답변: 지역 + 일련번호 3자리**

**Q6.** 담당자가 변경되거나 인수인계 시, 기존 현장코드는 그대로 유지됩니까, 아니면 새 코드로 변경됩니까?

> **답변: 새 코드로 변경**

---

### 4단계. 승인특가 작성

**Q7.** 승인특가 작성 시 기입해야 하는 "현장관련 사항"의 항목 목록을 알 수 있습니까?

> **답변: 현장코드만 있으면 됨**

**Q8.** 승인특가 작성 후 현장관리 프로그램에 자동 등록될 때, 어떤 항목이 자동으로 채워집니까?

> **답변: 현장코드만 관리**

**Q9.** 현장관리 프로그램에서 확인하는 특가금액·출고금액의 기준 시점은 언제입니까?

> **답변: 승인 시점**

---

### 5단계. SPEC OUT 처리

**Q10.** SPEC OUT이란 어떤 상황을 말합니까? 발생 조건을 설명해 주시겠습니까?

> **답변: SPEC 수주 실패 또는 취소된 경우**

**Q11.** 대체사유는 정해진 항목 중 선택하는 방식입니까, 직접 입력하는 방식입니까?

> **답변: 직접 입력**

**Q12.** "확정"은 누가 처리합니까? 확정 후 내용 수정이 가능합니까?

> **답변: 담당자가 처리** *(확정 후 수정 가능 여부는 추가 확인 필요)*

---

### 6단계. 현장관리 프로그램 써칭 기능

**Q13.** "기간별 현장 현장(①)" 항목이 오타로 보입니다. 정확히 어떤 내용을 조회하는 기능입니까?

> **답변: 기간별 현장 집계내역**

**Q14.** "기타(④)" 써칭 항목의 확정 내용이 있습니까?

> **답변: 없음 — 기타 항목 제외**

**Q15.** 써칭 결과를 엑셀로 다운로드하거나 인쇄하는 기능이 필요합니까?

> **답변: 필요 없음**

---

### 7단계. 관공서 영업PJ회의 보고양식

**Q16.** 보고양식에 들어가야 할 항목 목록을 알 수 있습니까?

> **답변: 현장명, 진행단계, 담당자, 특가 여부, 출고금액**

**Q17.** 기존에 사용 중인 보고 양식(엑셀, 문서 등)이 있으면 샘플로 제공해 주실 수 있습니까?

> **답변: 없음**

**Q18.** 출력 형태는 어떻게 됩니까?

> **답변: 화면 조회**

---

### 8단계. 공통 확인 사항

**Q19.** 사용자 권한은 어떻게 구분됩니까?

> **답변: 영업담당·관리자 전체 수정 가능** *(영업지원 권한 범위 추가 확인 필요)*

**Q20.** 개발 항목 중 우선순위와 희망 납기를 알 수 있습니까?

> **답변: 개발 시작 후 1개월 이내**
