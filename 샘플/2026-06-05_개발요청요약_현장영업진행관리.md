## ■ 요청 배경

현재 승인특가 작성과 현장 등록이 별도로 운영되어 데이터 연계가 되지 않고, SPEC OUT 처리 시 대체 정보 누락이 빈번함. 현장 영업 프로세스 전 구간을 시스템으로 통제하고, 현장별 영업 현황을 한 곳에서 확인할 수 있도록 관리 체계 구축이 필요함.

---

## ■ 확정 업무 흐름

```
자동 견적 → 현장 등록(현장코드 자동 부과) → 승인특가 작성
※ 자동 견적 미진행 시 승인특가 작성 불가 (시스템 차단)
```
